# Тестирование мобильного приложения Яндекс Метро (Android)

## Постановка задачи
Проведено тестирование мобильного приложения Яндекс Метро после рефакторинга кода. Цель — проверить изменения, убедиться, что функциональность приложения сохранена, и подтвердить готовность новой версии к релизу.

## Применяемые виды тестирования
- Функциональное тестирование изменений
- Регрессионное тестирование
- Smoke-тестирование 
- Тестирование установки и обновления приложения
- Проверка работы на разных версиях Android и устройствах

## Используемые инструменты
- Android Studio
- Logcat
- Google Таблицы


## Результаты
- Проверены изменения в коде и затронутая функциональность
- Проведено регрессионное тестирование
- Оформлен отчёт о тестировании
- Найденные баги зарегистрированы
